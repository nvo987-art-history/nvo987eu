# NVO987.eu – Agenda Culturel (Paris Open Data)

This repository contains the source code of the website:

**https://nvo987.eu**

It is a lightweight, static, privacy-friendly interface displaying a curated list of cultural events in Paris, based exclusively on official Open Data sources.

---

## ✅ Purpose of this project

This website provides an **independent automated interface** for consultation and synthesis of public cultural event metadata.

It is **NOT an official service** of the City of Paris.

The official source remains the only authoritative reference.

---

## 🔒 Privacy & GDPR Compliance

This website is designed to minimize GDPR and privacy risks:

- No cookies
- No advertising trackers
- No analytics tools (Google Analytics, Facebook Pixel, etc.)
- No user accounts
- No user database
- No forms collecting personal information
- No personal data is intentionally collected or processed by the website

No IP logging or tracking is intentionally performed by the website itself  
(**the hosting provider may automatically log technical access data such as IP address, user agent, or timestamps for security and operational purposes**).

The site only fetches a local JSON file (`events.json`) generated from public Open Data sources.

---

## 📌 Data Source (Open Data)

All event information displayed on this website comes exclusively from the official Open Data platform:

**Paris Open Data (Ville de Paris)**  
https://opendata.paris.fr/

Dataset used:

**"Que faire à Paris ? – Événements et activités"**
