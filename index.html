Mentions légales / conformité

Conformité RGPD, absence de cookies, limitation de responsabilité, droits et licences.

1. Éditeur du site

Le site NVO987.eu est édité par l’association culturelle NVO987. Les mentions légales officielles complètes (obligations légales françaises) sont disponibles sur : www.nvo987.fr.

2. Objet du site

Ce site propose une présentation automatisée d’événements culturels publics à Paris, à partir de données publiées légalement via la plateforme officielle Open Data.

3. Absence d’hébergement de contenu tiers

Ce site ne télécharge pas, ne copie pas, n’héberge pas et n’archive pas de contenus tiers (images, vidéos, documents, contenus complets). Il affiche uniquement des métadonnées (titre, dates, lieu, résumé court, lien officiel).

4. Données personnelles / RGPD

Ce site est conçu pour minimiser tout risque RGPD :

Aucune donnée personnelle n’est collectée, stockée ou analysée. Aucun compte utilisateur, aucune inscription, aucun identifiant permanent, aucune base de données utilisateur.

Si une source publique contient accidentellement des informations personnelles (email, téléphone, nom), celles-ci ne sont pas intentionnellement exploitées.

Toute demande de retrait peut être envoyée à : contact@nvo987.eu.

5. Cookies / tracking

Aucun cookie publicitaire, aucun pixel marketing, aucun outil de tracking et aucun outil d’analyse (Google Analytics, Facebook Pixel, etc.) n’est utilisé sur ce site.
